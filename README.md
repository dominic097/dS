## dS
 _A Data Structure Library in JS_
 
### To begin

 **1. Install it :**

```bash
	npm install ds --save-dev
```
**2 Require and use :**

```js	
    var LinkedList = require('../types/linkedList'),
    	link = new LinkedList();
    
    link.add(10);
    
```

### Included data structures

- Linked List
- BT Tree
